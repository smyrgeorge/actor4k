ActorRegistry

abstract class ActorRegistry(loggerFactory: Logger.Factory)

Represents an abstract registry for managing actor instances in an actor system.

The ActorRegistry is responsible for the lifecycle management, registration, retrieval, and shutdown of actors. It supports both locally managed actors and dynamic actor creation through factories. The registry ensures thread-safe operations on stored actor instances.
