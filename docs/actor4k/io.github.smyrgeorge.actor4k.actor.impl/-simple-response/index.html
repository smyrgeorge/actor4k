class SimpleResponse<T>(val value: T) : ActorProtocol.Response(source)

A generic implementation of a response within the actor protocol.

This class represents a concrete response type used for replies in the communication protocol, wrapping a value of a specific type. It extends the ActorProtocol.Response class.
