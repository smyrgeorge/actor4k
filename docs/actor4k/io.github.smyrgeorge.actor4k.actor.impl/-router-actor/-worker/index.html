Worker

abstract class Worker<Req : ActorProtocol, Res : ActorProtocol.Response>(key: String = randomKey("router-worker"), capacity: Int = ActorSystem.conf.actorMailboxSize, stashCapacity: Int = ActorSystem.conf.actorStashSize, onMailboxBufferOverflow: BufferOverflow = BufferOverflow.SUSPEND) : Actor<Req, Res> (source)

Represents an abstract worker in an actor-based system. The Worker class is designed to handle specific types of requests and generate appropriate responses. It extends the Actor class and provides additional functionality related to worker availability and task processing.

Parameters

Req

The type of request the worker processes, which must inherit from ActorProtocol.

Res

The type of response the worker produces, which must be a subtype of ActorProtocol.Response.

key

The unique identifier for the worker, generated by default using a random key if not provided.

capacity

The maximum size of the worker's internal message queue.

stashCapacity

Indicates the actor's stash capacity.

onMailboxBufferOverflow

Indicates the actor's behavior in case that the mailbox is full.
