fun <State> simpleActorOf(initial: State, key: String = Actor.randomKey(), capacity: Int = ActorSystem.conf.actorMailboxSize, stashCapacity: Int = ActorSystem.conf.actorStashSize, onMailboxBufferOverflow: BufferOverflow = BufferOverflow.SUSPEND, onActivate: suspend (SimpleMessage<SimpleResponse<State>>) -> Unit = {}, onShutdown: suspend () -> Unit = {}, onReceive: suspend (State, SimpleMessage<SimpleResponse<State>>) -> State): Actor<SimpleMessage<SimpleResponse<State>>, SimpleResponse<State>>(source)

Creates a simple actor with specified configuration and behavior.

This method constructs an actor that operates using a state-based model with custom lifecycle handlers. The actor responds to SimpleMessage<SimpleResponse<State>> messages and provides a SimpleResponse<State> upon processing.

Return

An actor that processes SimpleMessage<SimpleResponse<State>> messages and returns SimpleResponse<State> results.

Parameters

initial

The initial state of the actor.

key

A unique key for the actor. Default is a randomly generated key.

capacity

The maximum number of messages the actor's mailbox can hold. Defaults to the system-configured mailbox size.

stashCapacity

The maximum number of stashed messages the actor can hold. Defaults to the system-configured stash size.

onMailboxBufferOverflow

Behavior of the actor when the mailbox overflows. Default is BufferOverflow.SUSPEND.

onActivate

Lifecycle handler invoked when the actor is activated. Default is an empty lambda.

onShutdown

Lifecycle handler invoked when the actor is shut down. Default is an empty lambda.

onReceive

Handler invoked for processing incoming messages. Updates and returns the new state.
