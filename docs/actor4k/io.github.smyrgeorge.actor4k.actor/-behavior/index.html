Behavior

sealed interface Behavior<Res : ActorProtocol.Response>(source)

Represents a behavior outcome or operation that a system can process.

The generic type parameter Res extends the Response class, representing the response type linked to the behavior. This sealed interface allows modeling behaviors such as successful responses, errors, or operational directives.
