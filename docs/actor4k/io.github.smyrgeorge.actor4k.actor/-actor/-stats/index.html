Stats

@Serializable
data class Stats(var createdAt: Instant = Clock.System.now(), var initializedAt: Instant? = null, var triggeredShutDownAt: Instant? = null, var shutDownAt: Instant? = null, var lastMessageAt: Instant = Clock.System.now(), var receivedMessages: Long = 0)

Represents statistics related to the lifecycle and message handling of an actor.

This data class contains various timestamps and counters that provide insights into the operational state of an actor, such as when it was created, initialized, shutdown. The last time a message was processed, and the total number of messages it has processed.
